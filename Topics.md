## üöÄ JavaScript Fast-Track Learning Path

Chapter 1: Variables and data types
Understanding how to declare and use variables, as well as the different data types primitive (e.g. null, undefined, strings, numbers, booleans, bigint, symbol) and structural (e.g. object, function) in JavaScript.

### **Phase 1: Core Foundations (Must-know first)**

* Variables (`var`, `let`, `const`)
* Primitive data types
* Reference types (object, array, function)
* Type coercion & comparison (`==` vs `===`)
* `typeof`, `instanceof`

---

### **Phase 2: Control Flow (Execution thinking)**

* `if / else`
* `switch`
* `for`
* `while`
* `do...while`
* `break`
* `continue`
* `for...of`
* `for...in`
* Truthy & falsy values

---

### **Phase 3: Functions (VERY IMPORTANT)**

* Function declaration
* Function expression
* Parameters & return values
* Arrow functions
* Scope (global, function, block)
* Closures
* Call stack basics

---

### **Phase 4: Objects & Arrays (Daily-use JS)**

* Object literals
* Accessing properties
* Array methods:

  * `map`
  * `filter`
  * `reduce`
  * `find`
  * `some`, `every`
* Spread operator (`...`)
* Destructuring (object & array)

---

### **Phase 5: DOM (Browser JS)**

* `document.querySelector`
* `getElementById`
* Event listeners
* Event bubbling & capturing
* `classList`
* Creating & removing elements
* Form handling

---

### **Phase 6: Asynchronous JavaScript (CRITICAL)**

* Callbacks
* Promises
* `.then()` / `.catch()`
* `async / await`
* Error handling (`try / catch`)
* Event loop (high-level understanding)

---

### **Phase 7: ES6+ Essentials**

* Template literals
* Arrow functions (deep understanding)
* Default parameters
* Destructuring
* Modules (`import / export`)
* Classes (syntax only, not OOP-heavy)
* Promises (revision)

---

### **Phase 8: Debugging & Testing**

* Browser DevTools
* `console.log`, `console.table`
* Breakpoints
* Stack trace reading
* `try / catch`
* Basic unit testing concept (Jest idea)
