### Function Scope

* Variables declared **inside a function** are accessible **only within that function**
* Variables declared **outside a function** are accessible **inside the function**
* Each function creates its **own scope**
* `var` is **function-scoped**
* `let` and `const` are **block-scoped**
* Inner functions can access outer function variables (**scope chain**)
* Variables inside a function are **not accessible outside**
* Function scope prevents **global pollution**
* Variables are created when the function runs
* Variables are destroyed after function execution finishes
